## Payment Module for PrestaShop 1.7

Blue Media payment module for Prestashop 1.7.x

Installation and configuration instruction is available in `docs` directory.

### Requirements

Type       | Value
---------- | ------------------
PrestaShop | 1.7.0.0 - 1.7.7.6
